# Instruções

### Configurando o Config

Criar arquivo **.env** na raiz do diretório **capmaringa** com as seguintes informações.
```python
SECRET_KEY=
DEBUG=
GITHUB_KEY=
GITHUB_SECRET=
GOOGLE_OAUTH2KEY=
GOOGLE_OAUTHSECRET=
PASSWORDSMTP=
FACEKEY=
FACESECRETKEY=
LINKKEY=
LINKSECRET=
MAINTENANCE_PAGE_URL=
AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
```
